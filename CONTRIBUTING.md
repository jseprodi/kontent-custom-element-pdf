# Contributing

Thank you for your interest in contributing to the Kontent.ai PDF Editor Custom Element!

## Getting Started

1. Fork the repository
2. Clone your fork
3. Install dependencies: `npm install`
4. Create a branch for your changes: `git checkout -b feature/your-feature-name`
5. Make your changes
6. Test your changes: `npm run dev` and `npm run build`
7. Commit your changes: `git commit -m "Add your feature"`
8. Push to your fork: `git push origin feature/your-feature-name`
9. Create a Pull Request

## Code Style

- Follow the existing code style
- Use TypeScript for type safety
- Run `npm run lint` before committing
- Write clear commit messages

## Testing

Before submitting a PR, please ensure:
- The code builds without errors: `npm run build`
- The code passes linting: `npm run lint`
- The custom element works in Kontent.ai

## Questions?

Feel free to open an issue for any questions or suggestions!
